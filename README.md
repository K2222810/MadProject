# MadProject
An Android project built with Kotlin and Gradle.

## Description
MadProject is an Android application project that appears to be built with Kotlin and Gradle. It was a useful way to practice mobile app structure, Android project organisation, and working with a native app codebase.

This project helped strengthen my understanding of how Android apps are organised and built.

## Planning
Before building, the focus was likely on how the Android app screens, resources, and app logic would fit together.

Planning included:

- Organising app structure and resources
- Thinking about app flow and screens
- Planning Gradle and project setup
- Testing how the app components worked together

## Brief
The goal of this project was to build and practice an Android application using Kotlin.

The project needed to show stronger understanding of native app structure and Android development workflow.

## Build Process
The project started with setting up the Android project structure and Gradle configuration. From there, app files and resources were added and organised inside the project.

As the project developed, the focus moved to improving app structure, understanding native workflows, and making the project more complete.
